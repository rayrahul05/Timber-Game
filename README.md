🌲 Timber Game

A simple and engaging arcade-style game where the player chops down a tree while racing against time.

The player must continuously cut the tree while avoiding incoming branches. If a branch hits the player, the game ends. The challenge increases as the player must react quickly and choose the correct side to stay safe.

✨ Features:

⏱️ Time-based gameplay for added challenge
🌿 Dynamic tree with random branch positions
⚠️ Collision detection (player dies if hit by branches)
☁️ Animated background with 3 moving clouds
🐝 A bee that moves across different positions
🔊 Sound effects for actions and events

🎮 A fun project to demonstrate game logic, animations, and basic physics in a simple environment.
